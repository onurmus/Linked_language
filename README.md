# Linked Language
Linked Lanuage is a web site for learning languages using semantic relations of words. Two types of users exists namely feeders and learners. 

## Prerequisites
- JDK 1.8 or later
- Maven 3 or later
- MySQL 5.6 or later

## Stack
- Spring MVC
- Spring Security
- Spring Data JPA
- Maven
- JSP
- MySQL

## Run
```mvn jetty:run```
